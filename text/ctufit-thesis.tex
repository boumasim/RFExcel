\ctufitabstractCZE{Tato bakalářská práce se zabývá návrhem a implementací Python knihovny jakožto rozšíření pro Robot Framework, která má za účel testovat a modifikovat soubory různých tabulkových formátů. Knihovna má jasně strukturované uživatelské rozhraní, které je striktně oddělené od implementačních detailů. Vnitřní architektura aplikace využívá různých návrhových vzorů pro dosažení flexibility, jednoduché rozšiřitelnosti a možnosti obměny nízkoúrovňových knihoven, které používá pro interakci se soubory. Zároveň byl kladen důraz na streamovací mód, který konzumuje konstantní množství paměti a je optimální pro zpracování velkých souborů. Knihovna byla vyvinuta na základě požadavků zaměstnavatele a bude se používat právě pro testování obsahu tabulkových souborů. Knihovna bude vydána jako open-source, aby ji mohli využít i další uživatelé. Knihovna byla úspěšně navržena, implementována a~zdokumentována, zároveň vydána pomocí PyPI.}
\ctufitabstractENG{This bachelor's thesis focuses on the design and implementation of a Python library, developed as a Robot Framework extension for testing and modifying various spreadsheet file formats. The library features a clearly structured user interface that is strictly decoupled from its implementation details. Its internal architecture utilizes various design patterns to achieve flexibility, easy extensibility, and the ability to swap the low-level libraries used for file interaction. Furthermore, emphasis was placed on a streaming mode, which consumes a constant amount of memory and is optimal for processing large files. The library was developed based on the requirements of my employer and will be used specifically for testing the content of spreadsheet files. It will be released as an open-source project so that other users can utilize it. The library has been successfully designed, implemented, documented, and published via PyPI.}
\ctufitkeywordsCZE{Robot Framework Excel knihovna, tabulková knihovna, Python knihovna, open-source, PyPI}
\ctufitkeywordsENG{Robot Framework Excel library, tabular library, Python library, open-source, PyPI}
